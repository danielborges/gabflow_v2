# ADR-006 — PostgreSQL/PostGIS para análise territorial

## Status
Aceito

## Decisão
Utilizar PostGIS para armazenamento, indexação e consulta de coordenadas, bairros, regiões e áreas administrativas.

## Consequências
- suporte a mapas de calor;
- consultas por proximidade;
- agrupamentos territoriais;
- necessidade de governança de geocodificação;
- cuidado com precisão de endereços pessoais.
